# AI-Assistant-Streamlit-Groq-LangChain
A modern AI chatbot built using Streamlit, LangChain, and Groq LLMs with a clean white UI and smooth streaming response effect.
This project demonstrates how to build a real-time conversational AI system with memory, model selection, and downloadable chat history.
🧠 Features
🤖 Multiple LLM support (Groq models)
🎛 Adjustable creativity (temperature control)
💬 Chat memory using session state
🗑 Clear chat functionality
💾 Download conversation as text file
🎨 Clean white UI with custom CSS styling
⚡ Smooth streaming response effect

🔐 Secure environment variable handling with .env

🛠 Tech Stack

Frontend: Streamlit
LLM Provider: Groq
Framework: LangChain
Environment Management: python-dotenv
